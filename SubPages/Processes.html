Student Support Excel:

Make sure we are using the sign in Excel sheet to track what students come in. Please fill out as accurately as possible. The more information, the better!

“2026 Student Support” is the document we use. If you have any questions / issues with the sheet, contact Jacob!

  1.  Make sure you open the file in the Excel desktop app. The web version will not properly pull in the date / time when they scan in. 
  1. Make sure that you enable macros for Excel, this will again be critical for the date / time function. It should have a popup when you first open it asking if you trust the author and want to allow macros. If this doesn't show up, you can also manually enable them, here is a guide on doing that manually: https://www.indeed.com/career-advice/career-development/how-to-enable-macros-in-excel

Other things to keep in mind:

  • If you scan / enter an ID #, the name should automatically populate. If no name appears, either A.) that is not actually their ID # B.) they are not in IQ as of the time I imported the names (Weekly on Mondays)  or C.) the lookup formula got deleted out, so copy it down from a cell above that does have the name populating.
  • If you need to delete an entry, you can just clear out the ID # field, and it will remove the Name, time and date. 
  • For the "Issue" field, choose the most applicable issue, make sure to note difference between something having an issue and being broken. 

If yall have any other questions or if something stops working, feel free to reach out, and I can try and get it figured out!
